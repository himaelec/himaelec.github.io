東北大学電気情報系2023年8月実施【電磁気学】回答

(1) ガウスの法則を用いて電界を求める。

  1. \(r < a\) の場合: 導体球内部なので電荷は存在せず、ガウスの法則より、 \[\int_{S} \bm{E} \cdot d\bm{S} = \frac{Q_{in}}{\epsilon_0}\] ここで、\(Q_{in}\)は閉曲面内の電荷の総量である。\(r < a\) では \(Q_{in} = 0\) なので、 \[E(r) \cdot 4\pi r^2 = 0\] したがって、\(E(r) = 0\)。

  2. \(a < r < b\) の場合: 半径\(r\)の球面にガウスの法則を適用する。閉曲面内部には導体球1の電荷\(Q_1\)のみが存在するので、\(Q_{in} = Q_1\)。 \[\int_{S} \bm{E} \cdot d\bm{S} = \frac{Q_1}{\epsilon_0}\] 球対称性より電界は径方向を向くので、 \[E(r) \cdot 4\pi r^2 = \frac{Q_1}{\epsilon_0}\] よって、\(E(r) = \frac{Q_1}{4\pi\epsilon_0 r^2}\)。

  3. \(b < r < c\) の場合: 導体球殻内部なので電界は0。 したがって、\(E(r) = 0\)。

  4. \(c < r\) の場合: 半径\(r\)の球面にガウスの法則を適用する。閉曲面内部には導体球1の電荷\(Q_1\)と導体球殻2の電荷\(Q_2\)が存在するので、\(Q_{in} = Q_1 + Q_2\)。 \[\int_{S} \bm{E} \cdot d\bm{S} = \frac{Q_1 + Q_2}{\epsilon_0}\] 球対称性より電界は径方向を向くので、 \[E(r) \cdot 4\pi r^2 = \frac{Q_1 + Q_2}{\epsilon_0}\] よって、\(E(r) = \frac{Q_1 + Q_2}{4\pi\epsilon_0 r^2}\)。

(2) 電位\(V_a\)および\(V_c\)を求める。電位は無限遠を基準とする。 \[\begin{aligned} V_a &= -\int_{\infty}^{a} E(r)dr \\ &= -\int_{\infty}^{c} \frac{Q_1 + Q_2}{4\pi\epsilon_0 r^2}dr -\int_{c}^{b} 0 dr -\int_{b}^{a} \frac{Q_1}{4\pi\epsilon_0 r^2}dr \\ &= \frac{Q_1 + Q_2}{4\pi\epsilon_0} \left[ \frac{1}{r} \right]_{\infty}^{c} + \frac{Q_1}{4\pi\epsilon_0} \left[ \frac{1}{r} \right]_{b}^{a} \\ &= \frac{Q_1 + Q_2}{4\pi\epsilon_0 c} + \frac{Q_1}{4\pi\epsilon_0} \left(\frac{1}{a} - \frac{1}{b}\right) \\ &= \frac{Q_1}{4\pi\epsilon_0} \left(\frac{1}{a} - \frac{1}{b} + \frac{1}{c} \right) + \frac{Q_2}{4\pi\epsilon_0 c} \end{aligned}\]

\[\begin{aligned} V_c &= -\int_{\infty}^{c} E(r)dr \\ &= -\int_{\infty}^{c} \frac{Q_1 + Q_2}{4\pi\epsilon_0 r^2}dr \\ &= \frac{Q_1 + Q_2}{4\pi\epsilon_0} \left[ \frac{1}{r} \right]_{\infty}^{c} \\ &= \frac{Q_1 + Q_2}{4\pi\epsilon_0 c} \end{aligned}\]

(3) 導体球殻2のみを接地したとき(\(V_c = 0\))。 \(V_c = 0\) より、 \[\frac{Q_1 + Q_2}{4\pi\epsilon_0 c} = 0 \Rightarrow Q_2 = -Q_1\] このとき、\(V_a\)は \[\begin{aligned} V_a &= \frac{Q_1}{4\pi\epsilon_0} \left(\frac{1}{a} - \frac{1}{b} + \frac{1}{c} \right) + \frac{Q_2}{4\pi\epsilon_0 c} \\ &= \frac{Q_1}{4\pi\epsilon_0} \left(\frac{1}{a} - \frac{1}{b} + \frac{1}{c} \right) - \frac{Q_1}{4\pi\epsilon_0 c} \\ &= \frac{Q_1}{4\pi\epsilon_0} \left(\frac{1}{a} - \frac{1}{b} \right) \end{aligned}\] 静電容量\(C\)は、\(C = \frac{Q_1}{V_a}\)で定義されるので、 \[C = \frac{Q_1}{\frac{Q_1}{4\pi\epsilon_0} \left(\frac{1}{a} - \frac{1}{b} \right)} = \frac{4\pi\epsilon_0}{\frac{1}{a} - \frac{1}{b}} = \frac{4\pi\epsilon_0 ab}{b-a}\]

(4) 導体球1のみを接地したとき(\(V_a = 0\))。 \(V_a = 0\) より、 \[\frac{Q_1}{4\pi\epsilon_0} \left(\frac{1}{a} - \frac{1}{b} + \frac{1}{c} \right) + \frac{Q_2}{4\pi\epsilon_0 c} = 0\] \(Q_1\)について解くと、 \[Q_1 \left(\frac{1}{a} - \frac{1}{b} + \frac{1}{c} \right) = -\frac{Q_2}{c}\] \[Q_1 = -\frac{\frac{1}{c}}{\frac{1}{a} - \frac{1}{b} + \frac{1}{c}} Q_2 = -\frac{\frac{1}{c}}{\frac{bc - ac + ab}{abc}} Q_2 = -\frac{ab}{bc - ac + ab} Q_2\] \(V_c\)は \[\begin{aligned} V_c &= \frac{Q_1 + Q_2}{4\pi\epsilon_0 c} \\ &= \frac{-\frac{ab}{bc - ac + ab}Q_2 + Q_2}{4\pi\epsilon_0 c} \\ &= \frac{Q_2}{4\pi\epsilon_0 c} \left(1 - \frac{ab}{bc - ac + ab} \right) \\ &= \frac{Q_2}{4\pi\epsilon_0 c} \frac{bc - ac + ab - ab}{bc - ac + ab} \\ &= \frac{Q_2}{4\pi\epsilon_0 c} \frac{bc - ac}{bc - ac + ab} \\ &= \frac{Q_2}{4\pi\epsilon_0} \frac{b-a}{bc - ac + ab} \end{aligned}\] 静電容量\(C'\)は、\(C' = \frac{Q_2}{V_c}\)で定義されるので、 \[C' = \frac{Q_2}{\frac{Q_2}{4\pi\epsilon_0} \frac{b-a}{bc-ac+ab}} = \frac{4\pi\epsilon_0(bc-ac+ab)}{b-a}\]
